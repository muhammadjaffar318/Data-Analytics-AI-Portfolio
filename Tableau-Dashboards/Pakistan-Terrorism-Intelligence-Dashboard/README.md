# Pakistan Terrorism Intelligence Dashboard (1970 - 2020)

## Project Overview
This interactive Tableau dashboard provides a comprehensive intelligence analysis of terrorism incidents in Pakistan from 1970 to 2020. The dashboard helps identify terrorism trends, attack patterns, affected regions, terrorist groups, weapon usage, and casualties through advanced data visualization.

## Objective
The main objective of this project is to analyze terrorism activities in Pakistan, identify high-risk regions, monitor attack trends over time, and generate meaningful insights to support security analysis and policy decision-making.

## Tools & Technologies Used
- Tableau
- Data Visualization
- Data Analytics
- Business Intelligence
- Geographic Analysis
- Exploratory Data Analysis (EDA)

## Key Features

### Executive KPI Metrics
- Total Kills
- Total Injured
- Success Rate
- Average Kill per Incident

### Year-wise Terrorism Trends
- Analyze terrorist attack patterns from 1970–2020
- Identify peak terrorism periods
- Compare incident growth over time

### State-wise Impact Analysis
- Identify provinces/states with the highest casualties
- Compare terrorism impact across regions of Pakistan

### Attack Type Distribution
- Analyze most common attack methods
- Understand incident frequency by attack type

### Top Terrorist Groups Analysis
- Identify the most active terrorist organizations
- Compare group-wise attack activity and impact

### Weapon Type Analysis
- Analyze commonly used weapon categories
- Compare explosives, firearms, incendiary, and other methods

### Geographic Attack Map
- Visualize terrorism hotspots across Pakistan
- Identify high-risk areas using geospatial insights

### Interactive Filters
- State Filter
- Year Filter
- Month Filter
- Group Name Filter

## Key Insights
This dashboard enables analysts and researchers to:
- Identify terrorism hotspots in Pakistan
- Analyze long-term terrorism trends
- Monitor state-wise security risks
- Understand terrorist organization activity
- Support intelligence-based decision-making

## Business / Research Value
This dashboard can help security researchers, intelligence analysts, policymakers, and government institutions better understand terrorism patterns and make data-driven security decisions.
