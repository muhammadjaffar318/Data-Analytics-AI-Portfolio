# Honda Car Dashboard (Excel)

## Project Overview
This dashboard analyzes Honda car sales, revenue trends, and customer insights using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- KPI Dashboard

## Key Features
- Sales Analysis
- Revenue Tracking
- Performance Insights
- Interactive Dashboard
